# Senior Software Developer Portfolio

This is a modern, premium portfolio scaffold built with React + Vite + TailwindCSS. The design is inspired by Canva's clean, soft-gradient aesthetic.

Quick start

1. Install dependencies

```bash
cd /Users/arinjay/Projects/portfolio
npm install
```

2. Run locally

```bash
npm run dev
```

3. Build

```bash
npm run build
```

Deployment

- Vercel: Import the repo, set build command `npm run build` and output directory `dist`.
- Netlify: Use `npm run build` and set `dist` as the publish directory.
